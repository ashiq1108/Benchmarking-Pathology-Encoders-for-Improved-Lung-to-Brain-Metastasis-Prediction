# Benchmarking-Pathology-Encoders-for-Improved-Lung-to-Brain-Metastasis-Prediction
A computational pathology study benchmarking whole-slide image encoder pretraining strategies for brain metastasis risk prediction in NSCLC. This project evaluates how diverse pathology foundation models influence downstream Attention MIL performance on histopathology-based lung-to-brain metastasis prediction.
